# Comandos
## Para instalar el proyecto:
```bash
npm install
```

## Para inciar proyecto:
```bash
ng serve
```

## Para generar componentes es:
```bash
ng g component [Nombre del componente]
```

## Para generar componentes y saltar los tests:
```bash
ng g component [Nombre del componente] --skip-tests
```
